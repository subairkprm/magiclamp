# Brain HTTP API — surface map

Single source of truth for the routes exposed under `/api/v1/brain/*`. Each
route is implemented in a focused module under `brain/api/v1/_brain/` (see
[ADR 0001](../adr/0001-brain-router-split.md)).

| Method | Path | Module | Notes |
|--------|------|--------|-------|
| POST   | `/brain/memory/remember`         | `memory.py`    | Save a fact; auto-indexes to vector store when `RAG_ENABLED` |
| GET    | `/brain/memory/recall/{key}`     | `memory.py`    | Exact-key lookup |
| GET    | `/brain/memory/facts`            | `memory.py`    | All facts for the tenant |
| POST   | `/brain/memory/observe`          | `memory.py`    | Append an observation event |
| GET    | `/brain/memory/events`           | `memory.py`    | Filter events by `category` / `event_type` |
| GET    | `/brain/memory/stats`            | `memory.py`    | Counts of facts, events, training, decisions |
| POST   | `/brain/reason/lead`             | `reason.py`    | Async — returns a `task_id` |
| POST   | `/brain/reason/ask`              | `reason.py`    | Async — returns a `task_id`; result includes `retrieval_mode` & `citations` |
| POST   | `/brain/reason/ask/stream`       | `reason.py`    | **SSE** — see [streaming reference](streaming-ask.md) |
| POST   | `/brain/reason/decide`           | `reason.py`    | Async — returns a `task_id` |
| GET    | `/brain/reason/self-analyse`     | `reason.py`    | Synchronous self-analysis |
| GET    | `/brain/tasks/{task_id}`         | `reason.py`    | Status / result for any async task |
| GET    | `/brain/training/stats`          | `training.py`  | Sample count + export-readiness |
| POST   | `/brain/training/add`            | `training.py`  | Append a training pair |
| POST   | `/brain/training/export`         | `training.py`  | Admin-only — JSONL + Modelfile |
| POST   | `/brain/changes/record`          | `changes.py`   | Record a categorical change |
| GET    | `/brain/changes/history`         | `changes.py`   | Recent change events |
| GET    | `/brain/scheduler/jobs`          | `scheduler.py` | List background jobs |
| POST   | `/brain/scheduler/run/{job_id}`  | `scheduler.py` | Admin-only — manually trigger a job |

> The autogenerated OpenAPI schema lives at `GET /openapi.json` and renders
> at `GET /docs` when the API is running. The hand-curated map above is
> mirrored by `tests/test_brain_split.py::test_all_legacy_brain_endpoints_still_registered`,
> so the test suite fails if the surface ever drifts from this document.
